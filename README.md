Aquí tienes un archivo README.md profesional y detallado para tu repositorio, estructurado para que cualquier persona (o tu profesor) entienda perfectamente qué hiciste en esta Tarea #1.Gestión de Contactos - Programación de Interfaces Este proyecto consiste en una aplicación de escritorio desarrollada en Java utilizando la librería Swing. El objetivo principal es la gestión eficiente de contactos aplicando el patrón de diseño MVC (Modelo-Vista-Controlador) y cumpliendo con los requerimientos de optimización de la Unidad 1 de la Universidad Politécnica Salesiana.🚀 Funcionalidades Implementadas De acuerdo a la rúbrica de la actividad, se han integrado las siguientes mejoras avanzadas:Interfaz Organizada por Pestañas (JTabbedPane): Pestaña Contactos: Contiene el formulario completo de registro (Nombres, Teléfono, Email, Categoría, Favorito), búsqueda dinámica y la tabla principal.Pestaña Estadísticas: Espacio dedicado para procesos de exportación y monitoreo del sistema.Manejo de Eventos Avanzados: Eventos de Teclado: Implementación de atajos como la tecla ENTER para agilizar el guardado de contactos.Eventos de Mouse: Menú contextual (clic derecho) sobre la tabla para acciones rápidas como eliminar registros.Gestión de Datos y Visualización: JTable con Filtrado Dinámico: Visualización de contactos con capacidad de búsqueda en tiempo real mientras el usuario escribe.Exportación a CSV: Funcionalidad para volcar la base de datos de contactos en un archivo de formato plano .csv.Barra de Progreso (JProgressBar): Indicador visual que muestra el avance durante el proceso de exportación de datos.🏗️ Arquitectura del Proyecto (MVC) El proyecto sigue estrictamente el patrón Modelo-Vista-Controlador para garantizar la escalabilidad:modelo: Contiene la lógica de datos (persona.java) y el acceso a archivos (personaDAO.java).vista: Define todos los componentes gráficos y el diseño de la interfaz (ventana.java).controlador: Gestiona la interacción entre el usuario y el sistema, manejando todos los eventos (logica_ventana.java).🛠️ Requisitos e InstalaciónJava JDK: Versión 11 o superior.IDE: Eclipse (recomendado), NetBeans o IntelliJ.Estructura de Archivos: La aplicación guarda automáticamente los datos en la ruta C:/gestionContactos/datosContactos.csv.
